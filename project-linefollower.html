Line Follower Robot

Autonomous Sensor-Driven Robot

About the Project

A robot that autonomously follows a line on the ground using IR reflectance sensors. The sensors detect the contrast between the line and the floor surface, and the control logic adjusts motor speeds to keep the robot centered on the line.

The project focused on sensor calibration and reliable control logic to handle curves, intersections, and varying surface conditions. Built and programmed from scratch including a custom PCB designed in TraxMaker.
